Keshav's Portfolio v2
=====================

This is the source code for my personal portfolio website, built with React, TypeScript, and Tailwind CSS. It showcases my projects, skills, and provides a way to get in touch with me.

**Live Demo:** [https://keshavportfolio-nine.vercel.app/](https://www.google.com/search?q=https://keshavportfolio-nine.vercel.app/)

Features
--------

-   **Sleek UI/UX:** A modern, dark-mode design with subtle animations and effects, including a particle background, custom cursor, and a rocket page transition.

-   **Responsive Design:** Fully responsive layout that looks great on all devices, from mobile phones to desktops.

-   **Projects Showcase:** A dedicated "Work" page listing my projects with images, descriptions, tags, and links to the live demo and GitHub repository.

-   **Case Studies:** Detailed case study pages for each project, explaining the problem, solution, features, and technologies used.

-   **Dynamic Tech Stack:** An animated, scrolling "Tech Stack" component to showcase my skills.

-   **GitHub Activity:** Integrates and displays my GitHub contribution graph.

-   **Contact Form:** A functional contact form on the "Contact" page.

-   **Blog (Coming Soon):** A "Blog" page, currently with a "Coming Soon" placeholder, ready for future articles.

Tech Stack
----------

This project is built using a modern web development stack:

-   **Frontend:** React, TypeScript

-   **Routing:** React Router DOM

-   **Styling:** Tailwind CSS

-   **Animations:** Framer Motion

-   **Icons:** Lucide React

-   **Build Tool:** Vite

Project Structure
-----------------

The project is organized into the following main directories:

-   `src/pages/`: Contains the main page components (e.g., `Home.tsx`, `Work.tsx`, `About.tsx`).

-   `src/components/`: Contains reusable components (e.g., `Header.tsx`, `Footer.tsx`, `TechStack.tsx`).

-   `src/assests/`: Contains static assets like images.

Getting Started
---------------

To run this project locally:

1.  **Clone the repository:**

    Bash

    ```
    git clone https://github.com/your-username/portfolio_v2.git
    cd portfolio_v2

    ```

2.  **Install dependencies:**

    Bash

    ```
    npm install

    ```

3.  **Run the development server:**

    Bash

    ```
    npm run dev

    ```

    This will start the development server, typically at `http://localhost:5173`.

Available Scripts
-----------------

In the project directory, you can run:

-   `npm run dev`: Runs the app in development mode.

-   `npm run build`: Builds the app for production to the `dist` folder.

-   `npm run lint`: Lints the project files using ESLint.

-   `npm run preview`: Serves the production build locally for preview.
